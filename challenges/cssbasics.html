Inheritance in CSS

Sometimes, a property for an element is not specified. In these cases, if the property is an inherited property (for example color), the property's value will be set to the value that was sepcified for the parent element. In cases where the property is not inheritable, (for example flexbox properties), the value will be set to default.

Specificity in CSS

Specificity is basically a ranking system for different CSS rules. If multiple of these rules point to the same element and possibly conflict with each other, the specificities of the rules determine which rule is ultimately applied to the element. For example, inline styles are prioritized over paragraph styles.

ID and Class Attributes

The main difference between ID and class attributes is that many elements can be assigned to a single class, while an ID must be unique to one element. For example, if I had many buttons doing different things, each button would have a different ID (like "reload button" or "delete button"), but each button could also fall under the class "button", with consistent styling elements for each button.
