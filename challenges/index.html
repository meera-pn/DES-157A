Why we use the Normalize CSS Script

We use the normalize CSS script to allow consistency across different browsers. Different browsers have different default values for CSS properties, and if these default values are used, the webpage won't look the same across browsers. Normalizing overrides these defaults to keep the look of the webpage the same no matter what browser it is being viewed on.

What is Semantic Markup

Semantic markup communicated information about the content of a webpage that is more specific than the way it looks. For example, the look of a header could be created by just changing the font size/weight, but the "header" tag communicates that this is the title of the section. This has many uses, like allowing browsers to see what the main points of a webpage are and allowing screen readers to communicate information more effectively.
